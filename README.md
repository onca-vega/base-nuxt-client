# Base Nuxt Client

Sample repo for quick star using Nuxt 3

## Requirements

- Node installed (>= v9.11.2)

## Install dependencies

After cloning the repo, cd into folder and install dependencies:

```
cd folder/to/base-nuxt-client
npm install
```

## Run for development:

```
npm run serve
```

## Build:

```
npm run build
```
